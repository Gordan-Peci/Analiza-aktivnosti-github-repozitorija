# Analiza-aktivnosti-github-repozitorija
Cilj ovog projekta je izgraditi skladište podataka temeljeno na GitHub podacima te izraditi BI izvještaje koji omogućuju analizu popularnosti i aktivnosti repozitorija. Projekt obuhvaća izgradnju podatkovnog skladišta, provedbu ETL procesa i izradu vizualnih izvještaja u BI alatu.
