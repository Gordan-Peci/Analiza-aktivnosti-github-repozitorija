# Analiza-aktivnosti-github-repozitorija
Cilj ovog projekta je izgraditi skladište podataka temeljeno na GitHub podacima te izraditi BI izvještaje koji omogućuju analizu popularnosti i aktivnosti repozitorija. Projekt obuhvaća izgradnju podatkovnog skladišta, provedbu ETL procesa i izradu vizualnih izvještaja u BI alatu. Ovo je mali projekt s faksa di pokazujem znanje rada sa sql-om i skladištima podataka, te njihovom analizom. U predlošku se nalazi datoteka .docx koja detaljno opisuje projekt, sql datoteka u kojoj se nalazi sql upit korišten.
